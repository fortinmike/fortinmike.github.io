Illuminated Clock

Download

The ultimate clock screen saver for you Mac. The clock's design is a reproduction of the awesome Traser Black Storm Pro watch.

Features

Looks Nice

Yes, it looks nice. At least I think so!

Displays The Time

Another thing the screen saver is good at is displaying the time. Which is a good thing for a clock screen saver to be able to do.

Day / Night Cycle

The ambient lighting changes depending on the hour of the day so at night the clock face is barely visible and the dials stand out with a nice green glow. Check out the screenshots!
